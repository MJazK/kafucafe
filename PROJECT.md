# KafuCafe - Coffee Store Theme Project

## Project Vision

This is a **professional e-commerce store for a coffee and coffee tooling company** built on Shopify's Horizon theme. The store specializes in selling:
- **Premium coffee beans** (single origin, blends, roasted-to-order)
- **Coffee brewing equipment** (grinders, espresso machines, pour-over kits, accessories)
- **Coffee tooling** (scales, tampers, filters, cleaning supplies)

## Brand Identity

- **Brand Name**: KafuCafe
- **Vibe**: Professional, artisanal, warm, premium yet accessible
- **Target Audience**: Coffee enthusiasts, home baristas, specialty coffee lovers

## Product Categories

### 1. Coffee Beans
- Single origin coffees (by region: Ethiopian, Colombian, Brazilian, etc.)
- Blends (espresso blends, filter blends)
- Roast levels (light, medium, dark)
- Grind options (whole bean, ground)
- Subscription options

### 2. Brewing Equipment
- Espresso machines (manual, semi-auto, super-auto)
- Grinders (blade, burr - hand and electric)
- Pour-over equipment (V60, Chemex, Aeropress, French press)
- Drip coffee makers
- Cold brew systems

### 3. Accessories & Tools
- Coffee scales (with timers)
- Tamper & distribution tools
- Filters & paper
- Milk pitchers
- Cleaning supplies
- mugs & drinkware

## Design Direction

### Color Palette
- Warm, earthy tones (coffee-inspired)
- Primary: Deep browns, warm neutrals
- Accent: Coffee cream, caramel tones
- Clean whites for product imagery

### Typography
- Warm, readable fonts
- Professional headings
- Clean body text for product details

### Layout Priorities
- Showcase product quality with large imagery
- Clear product information (origin, roast level, flavor notes)
- Easy navigation between coffee and equipment
- Trust-building elements (reviews, sourcing info)

## Key Features Needed

1. **Product Collections** - Organized by product type and coffee origin
2. **Rich Product Pages** - Flavor notes, origin stories, brewing recommendations
3. **Subscription Options** - Coffee subscription service
4. **Equipment Guides** - Brewing guides for equipment
5. **Blog/Content** - Coffee education, brewing tips

## Technical Notes

- Built on Shopify Horizon theme
- Uses Liquid templating
- Vanilla JavaScript with Web Components
- Modern CSS with CSS variables
- Schema-driven configuration
